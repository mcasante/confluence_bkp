Marcado de Capitais : Melhorias no Controle de Debêntures

Created by Gabriel Sarmento Zamora on Jan 31, 2022

O nosso legado com as debêntures é muito ruim, seja o controle contábil das debêntures, como a estrutura de pagamentos, e o risco fiscal envolvido na operação.

Para emissões futuras, e para corrigir o que já temos, é importante fazermos diversas melhoras, as quais podemos classificar em dois grandes grupos:

  1. Melhoras estruturais

  2. Melhoras operacionais

Além do mais, é recomendável que usemos alguma sec de mercado, dado que não existe vantagem objetiva para nosso BP ter uma securitizadora.

Melhorias Estruturais

  1. Criar uma planilha de acompanhamento do 0

  2. Estruturar esta planilha, da máxima forma possível, junto com o time de dados para que a maior parte do preenchimentos seja automático desde o D+0

Melhorias Operacionais:

  1. Cashflow já roda hoje com todas as operações, mas é preciso ter certeza de que todo o código está correto

  2. Ter um local objetivo para comprar os relatórios de cada debênture, e se eles fazem sentido

  3. Arquivo CNAB automatizado

  4. Automatizar planilha de Estoque.

  5. Concentrar todas as comunicações no Hubspot, e fazer vários disparos de uma vez somente

  6. Criar um acompanhamento do Esperado x Realizado + projeção de retornos daqui pra frente
