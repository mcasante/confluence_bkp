Project: CX : Melhoria de Processos - Onboarding

Created by Fernanda Mano, last modified on Sep 22, 2021

0. TAREFAS:

  • Mapear processo que (possivelmente) podem ser aprimorados;
  • Mini-entrevistas com os envolvidos internos (visão interna de melhorias);
  • Solicitar pro Clinton se ele teve algum feedback em pós-venda sobre essa parte da esteira (visão externa); 
  • Estruturar um novo processo (possível) com as melhorias apontadas; 
  • Deixar em aberto percepções sobre melhorias mais estruturadas para o processo, que não será possível implementar no momento; 
  • Discutir o novo documento com todos os envolvidos; 
  • Corrigir o novo documento; 
  • Colocar o novo processo em prática; 

1. MAPEAMENTO INICIAL DOS PROCESSOS; 

  • Retirar o chat do processo - devemos ter uma ferramenta para conduzir processo e o chat para tirar dúvidas ocasionais; 

  • Instruções para preenchimento do cadastro na plataforma da a55 - antigas e novas telas de formalização; 

  • Instruções sobre os documentos pós-aceita de proposta que devem anexados; 

  • Tempo de esteira - qual é o tempo de esteira total em onboarding? qual é o tempo que o cliente demora para preencher o cadastro pós-aceite na plataforma? qual é o tempo levado para colher a documentação? qual é o tempo levado para analisar a documentação? qual é o tempo levado para corrigir a documentação? qual é o tempo que leva até o desembolso depois de que todos os documentos estão corrigidos? 

  • Checagem de documentos - como irá funcionar uma possível checagem de documentos? quais serão as vantagens? conseguimos estabelecer uma métrica dessa melhoria? 

  • Dificuldades - em que etapa os clientes possuem mais dificuldades? O que pode ser melhorado na visão do cliente? Podemos fazer uma pesquisa para colher impressões? 

2. MINI-ENTREVISTAS COM O TIME INTERNO

Possíveis entrevistados: Ranielly, Everton, Ana Neiva, Rodrigo e Gabriella. 

ROTEIRO: 

  1. O que na sua visão não é uma boa experiência para o cliente no processo de onboarding pós-aceite de proposta? 

  2. Quais são os principais gargalos vivenciados no seu dia-a-dia de trabalho? 

  3. Na sua visão, o que podemos melhorar para conceder uma melhor experiência para o cliente no processo de onboarding pós-aceita de proposta? 

  4. Quais são os pontos de melhorias que podem ser realizados/implementados para melhorar o seu dia-a-dia de trabalho? 

  5. O que são ações micros (sem grande investimento) que auxiliam no processo pro cliente e time interno no hoje?
