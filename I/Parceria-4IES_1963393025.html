Parcerias : Parceria 4IES

Created by Tech A55 on May 31, 2022

Dor que queremos resolver:

4IES é uma empresa que possui um software de gestão financeira voltando para instituições de ensino. Eles querem lançar um produto para garantir zero inadimplencia pra esses instituições, usando o crédito da a55.

Como faremos isso:

4IES nos encaminha (ainda de forma manual) os dados dos contratos que gostaria de garantir, para cada instituição de ensino

a55 devolve para 4IES proposta para garantir esses recebíveis, depositados em uma conta Escrow em nome do cliente final (instituição de ensino)

Caso cliente final tope, desembolso é realizado na conta vinculada. Após desembolso, valores depositados são transferidos para a 4IES, para que possam investir esse montante.

Daremos 2 meses de carencia, e a partir do terceiro mes de operação, 4IES passa a repassar esses recebíveis “garantidos” para a conta escrow, para que, mes a mes, realizemos a amortização das parcelas
