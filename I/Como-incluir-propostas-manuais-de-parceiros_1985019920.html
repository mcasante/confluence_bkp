Parcerias : Como incluir propostas manuais de parceiros

Created by Tamila Hohlenverger on Jun 24, 2022

Quando existir a necessidade de criar alguma proposta que não foi gerada pelo motor de pricing, poderá seguir o passo a passo a seguir:

  1. Abrir uma task para o time técnico rodar uma proposta para o CNPJ específico no nome do parceiro (o parceiro deve estar cadastrado na API). Modelo de task: RA-229

  2. Alterar dados da proposta no midgard e pedir para mesa de crédito aprovar;

  3. Verificar se deal foi gerado corretamente.

Existe uma task para que esse processo seja disponibilizado na seção de Parcerias: Geração de propostas no midgard. RA-233
