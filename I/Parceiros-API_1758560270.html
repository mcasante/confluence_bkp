Parcerias : Parceiros API

Created by andreia.brizolla, last modified on Aug 16, 2021

A integração de dados via API tem como objetivo trazer maior escalabilidade na originação de novos negócios.

Através da Credit API a55 nosso parceiros podem oferecer serviço de crédito dentro da sua plataforma, trazendo confiabilidade e praticidades para os seus clientes.

Passo a passo para integração dos parceiros:

  1. Envio do contrato padrão e posterior assinatura de contrato de parceria;

  2. Apresentação do fluxo de integração e como disponibilizamos os serviços de crédito na plataforma do parceiro PROTÓTIPO TELAS PARCEIROS;

  3. Após efetivada a parceria será realizado o cadastro do parceiro para liberação da chave de acesso da API. Para a realização do cadastro serão necessário os seguintes dados (atualmente o cadastro é feito via backend):

    1. CNPJ

    2. Razão Social

    3. Nome Fantasia

    4. Atividade

    5. E-mail

    6. Telefone

    7. Nome do Responsável

    8. URL plataforma

  4. Os dados devem ser registrados em Cadastro Parceiros;

  5. Disponibilizar as chaves em prod e stg;

  6. Disponibilizar o Guia de Integração

  7. Disponibilizar a Doc API

  8. Marcar um call de alinhamento da implementação da integração junto ao time de produto e tech do parceiro.
