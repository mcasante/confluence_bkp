Parcerias : CREDIT API a55

Created by andreia.brizolla, last modified on Jun 08, 2021

Com a Credit API a55 os parceiros da a55 pode ofertar soluções de crédito integrado aos serviços das suas plataformas.

Modelos de propostas:

Propostas em Lote: Este é um recurso para que os Parceiros possam enviar dados prévios de um lote de clientes e obter uma proposta de crédito pré-aprovada, que poderá ser divulgada no canal que seja mais aderente ao seu público. Os dados podem ser enviados anonimizados ou não a depender da política de privacidade da sua plataforma.

Propostas unitária: Com este recurso, os clientes do parceiro podem solicitar uma proposta de crédito dentro da plataforma parceira, seus dados são compartilhados com a a55 para geração de uma proposta customizada que é disponibilizada via plataforma parceira, após o aceite o cliente é direcionado para a plataforma a55 para formalizar o crédito.
