Plataforma de Dados : Propostas aceitas

Created by Bianca Sofia on Apr 19, 2022

As propostas aceitas se referem à contagem das propostas para clientes qualificados que foram aceitas, ou seja, que possuem os requisitos necessários para o desembolso, sendo que são considerados os clientes que não são MEIs (Microempreendedor individual que se considera empresário), que tenham tempo de fundação superior à 6 meses consecutivos e não sejam parceiros com origens em : API Integration Partner, Agency Partner, Formulário parceria, Integration Partner, New Partnership, BNDEs (campanha), BNDEs (fluxo padrão)).
