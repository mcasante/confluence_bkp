Plataforma de Dados : Lifetime Value (LTV)

Created by Alessandro Oliveira (Unlicensed), last modified on Mar 15, 2022

O LTV é uma métrica utilizada para medir o quanto de lucro um cliente gerou para a empresa durante um intervalo de tempo. Ou seja, na prática representa os valores amortizados por cliente da a55 durante um período.

É uma métrica a ser comparada principalmente com o CAC (CAC - Custo de Aquisição de Clientes), de modo que idealmente o LTV deve ser maior que o CAC para representar lucro para a companhia.
