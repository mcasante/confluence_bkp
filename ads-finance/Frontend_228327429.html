Landing Page

Wordpress

Passo a passo com o setup da configuração do wordpress para desenvolvimento na Landing Page.

Requerimentos

  • Arquivos

    • .htaccess

    • .env

    • .pem

    • .sql (dump)

  • Pacotes

    • PHP 7.2.31

    • Composer 1.10.8

    • Node

    • Yarn

    • Servidor PHP (Apache, NGINX ou XAMPP)

    • Banco de dados mySQL

Instalação

  • Configurar o banco de dados mySQL local

mysql -u root -p

→ Em caso de erro no sock, crie um link simbólico com o caminho do sock:

sudo ln -s /Applications/XAMPP/xamppfiles/var/mysql/mysql.sock /tmp/mysql.sock)

→ Crie o banco de dados de acordo com as informações de ambiente especificadas no arquivo .env.

→ Para conectar no servidor você deve possuir o arquivo com a chave de conexão (.pem).

# Conectar no servidor (deve estar no mesmo diretório da chave)
ssh -i a55_br.pem ubuntu@18.229.98.188

# Ganhar privilégios de superuser
sudo su

# Criar arquivo .sql no servidor
cd /home/ubuntu
mysqldump wp > db_backup.sql

# Copiar do servidor para local o arquivo .sql
scp -r -i a55_br.pem ubuntu@18.229.98.188:/home/ubuntu/db_backup.sql /

→ Importar arquivo gerado para dentro do banco previamente criado (phpMyAdmin).

  • Clonar o projeto do site-institucional dentro do diretório do servidor PHP local

git@github.com:access55/site-institucional.git

  • Instalar as dependências da raiz do projeto

composer install

  • Instalar as dependências do tema

cd /web/app/themes/a55-theme
composer install
yarn install
yarn build

  • Colocar o arquivo .htaccess no diretório web

  • Copiar o diretório com as imagens (uploads) do servidor para local

scp -r -i a55_br.pem ubuntu@18.229.98.188:/var/www/html/source_code/web/app/uploads var/www/html/source_code/web/app/uploads

  • Permissão no diretório de uploads

chmod -R 777 web/app/uploads

Acessos

  • Para acessar a Landing Page

http://localhost/a55/site-institucional/web/

  • Para acessar o ambiente WP Admin

http://localhost/a55/site-institucional/web/wp/wp-admin/

  • Solicitar usuário com acesso ao WP Admin
