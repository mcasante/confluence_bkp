DataHub : Manual da API DataHub

Created by Geison Voga, last modified on Jun 22, 2020

Este manual descreve a API de análise da plataforma Data Hub A55

A API Data Hub esta dividida em 5 grandes blocos de análise, sendo estes:

  1. Analise de dados públicos de empresas e perfil do mercado em que a empresa se encontra (em construção)

    1. Dados da receita Federal

    2. Analise de mercado competidos por meio do CNAE

    3. Geolocalização (estado e cidade) do mercado concorrente

  2. Análise estatística descritiva do faturamento:

    1. Cálculos estatísticos do faturamento ordenados mensalmente

    2. Análise comparativa da empresa com relação a base de dados de mercado de adquirência da Cielo

    3. Análise de distribuição de faturamento global, semanal e mensal

  3. Modelagem de previsão para faturamento analisando padrões de tendencia e sazonalidade

Estatística comparativa

Este serviço faz uma comparação entre o faturamento médio da empresa e o mercado de adquirencia Cielo. O resultado é uma comparação de desempenho global (toda a série historia) ou mensal (comparacao mês a mês). Por meio desta analise o cliente pode visualizar a media do faturamento, a media do mercado, qual o percentual de empresas que faturam menos que a sua média e a sua variação relativa em relação a media do mercado.

Estatística comparacao

Tipo de comunicação

Tipo de Conteúdo

Distribuição descritiva

Este serviço apresenta ao cliente 3 forma de apresentação da distribuição do faturamento, sendo estas: global, semanal e mensal. Na distribuição global é possível visualizar a estratificação de todos valores registrados no bando de dados para o cliente, esta analise permite a identificação do ticket médio de faturamento e quão dispersa é a faixa de faturamento

Serviço

Estatística comparacao

Tipo de comunicação

HTTP

Tipo de Conteúdo

JSON

url

3.235.17.104:5000/distribuicao

Método

POST

Parametros

id:”numero de cliente na base da a55”

período: {“global” ou “mensal”}

propriedade:{media, media_max, media_mim}
