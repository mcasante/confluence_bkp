Plataforma a55 : Companies registration flow

Created by Tech A55 on Aug 31, 2021

Purpose of the feature:

The purpose of this flow is to collect the data of the company that is applying for credit, which is received in the registration form (plataforma.a55.tech/cadastro) and create the record of that company in our database.

Data obtained throughout form registration:

-TaxId

-Average monthly revenue

-Contact data

It is noteworthy that this process does not only use the data provided by the customer, but also consultations with external services that provide additional data about the company, using their tax-id as input.

Data obtained throughout external API (BigDataCorp: https://bigdatacorp.com.br/ )

-Naic code

-Foundation date

-Corporate name

-Fantasy name

-Address

-Social Capital

-TaxRegime
