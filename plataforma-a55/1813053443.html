Plataforma a55 : Integração Bling (ERP)

Created by Tech A55 on Oct 04, 2021

O que é a bling?

O Bling é um sistema de gestão empresarial (ERP) para micro e pequenas empresas.

As funcionalidades da ferramentas possibilitam o controle sobre as vendas, finanças, estoque, produtos, clientes, pedidos, comissões de vendedores e serviços,

O sistema permite também a emissão de notas fiscais eletrônicas, a geração de boletos bancários, cobranças e propostas comerciais.

Além disso, é integrado com SIGEP Web dos Correios e com as principais plataformas de e-commerce.
