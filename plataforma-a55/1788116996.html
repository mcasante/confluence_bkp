Plataforma a55 : Bench-mark: Gestão de KPIs

Created by Tech A55, last modified on Sep 17, 2021

Principais KPIs que pretendemos disponibilizar via relatórios:

1 MRR / ARR

2 Custo de aquisição de cliente (CAC)

3 Tempo de retorno do CAC

4 Ticket médio

5 Lifetime value

6 Usuários ativos mensais ou usuários ativos por dia

7 Churn

8 NPS (Net Promoter Score)

Empresas que já fazem isso:

  • BareMetrics https://baremetrics.com/ : Software voltado para empresas SaaS, traz de forma automática os seguintes indicadores: ➔ Faturamento recorrente mensal

    ➔ Net revenue

    ➔ Cobranças falhas

    ➔ Lifetime value

    ➔ Lifetime value por plano

    ➔ Reembolsos

    ➔ Upgrades

    ➔ Churn de usuários

    ➔ Crescimento de clientes

    ➔ Contas canceladas

    ➔ Downgrades 

Fonte de dados: Múltiplas, integrações com processadoras de pagamento (Chargify, Stripe) e também uploads manuais de planillhas
