Parcerias : AS IS - Fluxo Jornada do Parceiro

Created by Rejane Ribeiro, last modified on Feb 16, 2021

Atualmente , o fluxo de parceiro deverá em vários passos ser executado de forma manual.

No modelo de parceria para geração de leads direcionados ou seja quando houver o compartilhamento de dados , então o parceiro deverá enviar de forma manual os dados dos clientes em formatos (.xls, .csv, .xml ou um json).

Esses dados deverão ser carregados em uma base para o time de "pricing” consumir e gerar as propostas indicativas. Essas propostas serão retornadas para os parceiros e assim divulgadas para os potenciais clientes.
