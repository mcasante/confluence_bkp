Parcerias : Integração de Dados dos Parceiros

Created by Rejane Ribeiro, last modified on Feb 16, 2021

Todo o fluxo da Jornada dos Parceiros deverá ser suportado pelas integrações disponibilizadas na API Parceiros A55.

Entretanto para um melhor entendimento dos dados transacionados entre a A55 e os Parceiros será necessário validar com Parceiros a possibilidade de compartilhar a base antecipadamente ao impulsionamento para segmentação da audiencia e geração de propostas indicativas.

Nesse caso, definimos que os dados mínimos que precisamos para efetuar a segmentação são:

  • Dados mínimos para alimentar API:

    • Periodicidade: Diária, por transação

    • Histórico: mínimo 12 meses

    • Dados transacionais de cartão:

      • id

      • adquirente

      • login

      • usuário

      • id_transacao

      • data_op

      • data_pg

      • valor_bruto

      • valor_liquido

      • valor_tarifa

      • valor_desconto

      • valor_rejeitado

      • valor_encargo

      • valor_credito

      • valor_extra

      • n_parcela

      • total_parcela

      • tipo_pg_sacado

      • tipo_pg_sacado_adquirente

      • tipo_pg_adquirente

      • status

      • status_adquirente

      • cliente

  • Exemplos de dados complementares (para um envio pontual inicial, não para serem integrados via API)

    • Perfil da base de cliente do seller (abertura por cliente): hábitos de compra (frequencia, produto, ticket médio, etc), forma de pagamento, gastos com frete, etc 

    • Cesta de produtos do seller: #produtos, ticket médio, produtos mais vendidos, margem por produto, faturamento por produto, etc

    • Margem do seller: % repasse para fornecedores, custo de produto, taxas pagas para adquirentes/integradores/plataformas, 

    • Crédito: valores adiantados por adquirente, taxa de adiantamento (juros), operações ativas e históricas com split, etc


  • Dados complementares requisitados para parceiros:

  • AVEC:

  1. Dados básicos

  2. GMV total do salão

  3. Recebimento líquido do salão por transação: Receita (-) de comissão aos profissionais (-) fees Avec

  4. Data de entrada do salão na AVEC 

  5. Ticket médio do salão*

  6. # profissionais*

  7. turn-over de profissionais*

  8. ticket médio por profissional*

*é possível termos mensalmente? se não na frequência possível
