Integração dados parceiros: Proposta unitária

O modelo de PROPOSTA UNITÁRIA tem como objetivo integrar a base de dados dos parceiros para geração de propostas de crédito personalizadas para seus clientes.

Documentação API PROPOSTA UNITÁRIA

Fluxo Principal Proposta

Descrição:

Neste modelo, os usuários terão uma área de acesso a crédito da A55, dentro da plataforma parceira. Ao clicar em solicitar uma proposta os dados do usuário serão compartilhados para partnership_api para análise e geração de proposta.
