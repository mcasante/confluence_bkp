Guilherme Pettene Dantas : Plataforma Parceiro"Pastinha"

Created by Guilherme Pettene Dantas, last modified on Aug 05, 2022

Contexto:

O pastinha é um parceiro que possui relacionamento com diversas empresas e fica encarregado de buscar alternativas de crédito para elas. O pastinha pode ser um ex-gerente de banco, uma boutique de M&A, um escritório de contabilidade, dentre outros.

Atualmente, temos 23 players desse tipo originando, com uma média de 5 leads por parceiro. Além disso, temos 60 parceiros desse tipo no pipeline de prospecção.

Para mais infos: Product Platform for Partners

Épico: Construir uma área logada para o parceiro pastinha.

Como parceiro da a55

Quero uma área logada

Para gerenciar meus clientes

User Story 1:

Como parceiro

Quero cadastrar uma empresa

Para fornecer crédito ao meu cliente

Critérios de Aceite:

Dado que temos os dados do cliente

quando o parceiro percorreu a jornada

conseguimos gerar uma proposta para o cliente do parceiro

User Story 2:

Como parceiro

Quero acompanhar o andamento das operações cadastradas

Para atualizar meu cliente

Critérios de Aceite:

Dado que temos uma operação cadastrada

quando o parceiro percorreu a jornada

conseguimos atualizá-lo sobre em qual etapa o cliente se encontra

User Story 3:

Como parceiro

Quero enviar a proposta para o cliente

Para que ele aceite a operação

Critérios de Aceite:

Dado que temos uma proposta

quando a mesa de crédito aprova o cliente

conseguimos enviar o contrato de formalização para o cliente final

User Story 4:

Como parceiro

Quero percorrer a esteira de formalização

Para que ocorra o desembolso da operação

Critérios de Aceite:

Dado que o parceiro percorreu a esteira de formalização

quando o cliente aceitou a proposta

conseguimos fazer o desembolso da operação

User Story 5:

Como parceiro

Quero saber a minha comissão mensal

Para gerenciar minhas receitas e despesas

Critérios de Aceite:

Dado que temos um % de comissão para cada parceiro

quando a operação é desembolsada

conseguimos informar ao parceiro o quanto ele vai receber no próximo mês
