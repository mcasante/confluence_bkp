Guilherme Pettene Dantas : Catalogar o produto das operações

Created by Guilherme Pettene Dantas on Sep 22, 2022

Tarefa: Adicionar uma coluna na tabela de contratos no power bi com o produto de cada operação

Os contratos podem pertencer a dois produtos:

“BR Dilution” ou “BR Clean”

Contratos BR Dilution:

  • Todas as operações com vehicle = “FIDC A55 SAAS” com loan date anterior à 01/05/2020

  • Todas as operações com vehicle = “FIDC A55 SAAS” posteriores a 01/05/2020 com Gross amount >= 500.000

  • Todas as operações com vehicle = “BV” cuja empresa já tinha uma linha pretérita ativa ou não com a a55.

  • Todas as operações com CNPJ (Taxid) = 15562467000100.

  • Todas as operações com vehicle = “FUNDO RAÍZES”

  • Todas as operações cujo operation code contenha “Laser Fast”

  • Todas as operações que contenham “BRIDGE” no operation code

  • Todas as operações com vehicle = “BONDS 5 SERIES 1”

  • Todas as operações com vehicle = “BONDS 6 SERIES 1”

  • Todas as operações com vehicle = “BONDS 7 SERIES 1”

Contratos BR Clean:

  • Restante das operações que não foram catalogadas como BR Dilution
