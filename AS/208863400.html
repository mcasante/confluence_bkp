Planilha com os campos das 3 APIs que utilizamos no processo:

As duas primeiras APIs deverão ser desenvolvidas pela fintech de acordo com os campos definidos pelo BNDES na planilha. Para o acesso à nossa API de atualização de status, será necessário a criação de um usuário na loja de APIs do BNDES. Segue um passo a passo para criação no ambiente de homologação:

a) Fintech acessa a loja de APIs do BNDES (https://apis-h.bndes.gov.br/store/) e cadastra seu usuário

Instruções disponíveis em: https://developers.bndes.gov.br/forum/category/6/portal-de-apis

b) Após essa criação, as fintechs devem informar o usuário criado respondendo a esse email. Nesse momento, o BNDES dará as permissões necessárias para acesso à API.

c) Somente após as permissões concedidas, a fintech poderá criar uma aplicação na loja.
Nesse momento as seguintes ações serão executadas:

  • criar uma aplicação

  • gerar token

  • fazer a associação da aplicação com a API "Canal MPME - 1.4".

https://developers.bndes.gov.br/forum/topic/45/como-assinar-uma-api-publicada-no-portal

d) Fazer uma chamada a API usando Postman/Insomnia
https://developers.bndes.gov.br/forum/topic/22/chamada-de-api-utilizando-o-postman

e) Tem mais alguma dúvida sobre a loja de APIs?
Publique sua dúvida no nosso fórum: https://developers.bndes.gov.br/forum/

  1. Planilha com os parâmetros que precisamos configurar no Canal para habilitação de fintech.

 Esta planilha deve ser preenchida e devolvida ao BNDES, assim que a fintech possuir as definições. Sobre esta planilha:

  • A descrição solicitada no primeiro quadro à esquerda será exibida juntamente com a oferta da fintech para o cliente durante sua jornada do Canal e deverá ter no máximo 140 caracteres.

  • As URLs solicitadas logo abaixo são as que chamaremos na jornada do cliente no Canal (Oferta e Proposta). Solicitamos os endereços em homologação e produção.

  • Na direita, os parâmetros solicitados definirão o público com quem a fintech atuará e dentro de que faixas (de tempo de constituição, faturamento e financiamento solicitado).
