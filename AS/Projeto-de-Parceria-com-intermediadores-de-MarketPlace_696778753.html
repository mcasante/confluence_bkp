Overview da solução

Estamos com a iniciativa de gerar cada vez mais oportunidades de financiamento através de empresas parceiras, principalmente intermediadores de MarketPlace. A ideia é que o cliente final dos nossos parceiros tenha acesso a soluções de crédito através da plataforma do nosso parceiro. Para isso, ele terá que seguir o seguinte fluxo descrito abaixo:

Etapas até proposta indicativa:

  1. Plataforma anuncia através de sua plataforma soluções de crédito da a55

  2. Cliente do parceiro deve aceitar nossos termos e condições de utilização dos seus dados

  3. Precisaremos receber via API o faturamento mensal estimado desse cliente e CNPJ

  4. Retornaremos via API na plataforma do parceiro uma proposta indicativa, segundo após o imput dos dados de CNPJ e faturamento

  5. Cliente do parceiro aceita proposta indicativa 

Etapas até geração de proposta firme

  1. Após aceite da indicativa, cliente deve nos dar acesso aos seus dados de sua conta do Google Ads via API (já temos uma integração que precisará ser revisada)

  2. Dados serão armazenados em uma base de dados a55, e atualizados diariamente de forma automática (não faremos uma "seleção desses dados", armazenaremos tudo que estiver disponível no GAds do cliente em nossa base)

  3. Time de princing/ds consumirá os dados armazenados do DB de forma automática para gerar propostas firmes

  4. Após vincular sua conta de google Ads, cliente/parceiro deverá nos dar acesso aos dados de faturamento desse cliente final (via api com adquirentes ou api direta com base de dados do parceiro - ex. Olist tem armazenado o histórico de faturamento de seus clientes)

  5. Proposta firme será gerada automaticamente com base nos dados de Google Ads e de faturamento, e ficará disponível em nossa plataforma

Etapas após aceite da proposta firme 

Cliente receberá email automático da a55 para confirmação do seu email, e com instruções para login na plataforma e acesso a proposta gerada
