Plataforma a55 : Formalization API

Created by Rafael Braga, last modified on Jul 04, 2022

  1. Resumo executivo: API para agrupar todas as chamadas de Formalização, desde proposta aceita até desembolso, pensando em escalabilidade de diferentes cliente e produtos para determinada etapa do fluxo.

  2. Contexto e Problema que queremos resolver:

    1. Automação de análises e respostas para suporte dos analistas de Formalização e a possibilidade de automação para diferentes tipos de pipes, relacionando tipo do produto/cliente X volume da proposta

  3. O que já foi descoberto:

    1. Atualmente as chamadas que são feitas na etapa de Formalização não estão centralizadas em um só local.

    2. Teremos diferentes tipos de pipes dentro da etapa de formalização que serão definidos conforme produto, cliente e volume.

  4. O que sabemos e o que não sabemos:

    1. O que não sabemos:

      1. Quais são os diferentes pipes

    2. O que sabemos:
