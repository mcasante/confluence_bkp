Plataforma a55 : Comite de arquitetura

Created by Tech A55 on Jul 15, 2020

Principais responsáveis pela arquitetura de sistemas: Kátia, Phill, João e Alfredo

 

Escopo de atividades:

 

  • Dump de dados para staging tanto para a realização de testes quanto para apresentações da plataforma (escolher 3/4 empresas que serão replicadas com os dados mascarados em staging)

 

  • Gestão dos ambientes aws (restringir quem poderá realizar alterações)

 

  • Sistema de monitoramento (prometheus/grafano) de sistemas para identificar gargalos

 

  • Controle de produtos que estarão dentro do aSR

 

  • Remodelagem sistema de permissionamento (baseado em perfis e papéis) – usar modelo do Django?

 

  • Nova gestão do processamento dos scrapers (montar um datahub de integrações no qual o desempenho de todas as APIs serão monitoradas)

Alternativa seria utilizar plataforma do Kafka (plataforma Open Source de processamento de dados) pra montar o datahub

 

  • Gestão de acessos ao banco de dados

 

  • VPN pros Scrapers

 

  • DB isolado para banking

 

  • Preparação para a iso 27001

 

  • Pen test

 

  • Documentação End-Points (prática, explicando o contexto no qual são utilizadas)
