Project: CX : CS a55

Created by Clinton Consul on Oct 13, 2021

As empresas buscam recursos financeiros no mercado e, para seu crescimento, não querem muitos credores mas sim poucos e de confiança.

Entenda que falamos de crédito todo dia mas para o cliente existe muita coisa em jogo - um projeto, uma admissão, pagar ou não uma conta, dar ou não um passo no negócio.

  • Tenha empatia,

  • Alinhe expectativas,

  • Pergunte detalhes das operações - é bom para seu desenvolvimento pessoal, para desenvolvermos soluções na plataforma

  • Conheça um pouco sobre operações em SaaS e ecommerce para conversar a altura com os clientes. Caso não saiba de algo, pergunte e os clientes vão ensinando!

  • Traga visibilidade sobre as dores e necessidades dos clientes para dentro da casa. Gestão em C-level é MUITO comprado com isso

  • Brigue pelos seus clientes. C-level é comprado, temos uma área de CX e Vendas que dá atenção ao cliente. Você não tem nada a perder.

  • Atenção à voz do cliente dentro da cia. CS é uma das áreas que mais entende do cliente e deve levar isso adiante.

Planos futuros:

  • estruturar trilhas de conhecimento para facilitar o crescimento dos empreendedores

  • definir uma periodicidade para contatos de acompanhamento

  • Pensar em estruturas de apresentar clientes para clientes (ecossistema de crédito fechado)

  • Fluxo de report estruturado com observações realizadas pela área - quando aumentou a receita, numero de vagas de trabalho criadas após o credito, novos projetos nos clientes,
