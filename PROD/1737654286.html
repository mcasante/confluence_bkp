Projetos sem Esquadra

Projeto sem esquadra é aquele que não possui o arquivo CODEOWNERS. Todos os engenheiros podem mergear. Sugere-se solicitar um Code Review no chat.

Novos projetos

Sugere-se que um novo projeto pertença a esquadra do engenheiro que faz o primeiro commit.

Contexto

Alinhar que os serviços de Banking vão ser distribuídos entre os times que têm propriedade sobre determinadas ações que o serviço trata.

Decisões

  • Banking (API, Agent e Scrappers)

    • Leitura de dados → Pricing

    • Cobrança e emissão de boletos → Ops

  • DataHub

    • Leitura de dados → Pricing (longo prazo)

  • Impacto da Plugg

    • Plataforma acredita que o momento de mudança é após terminarmos integração com Plugg

  • Roadmap de handover

    • Começamos com a passagem para Operações

    • Num segundo momento passaríamos para Pricing

  • Seria interessante cada squad separar um tempo da sprint para escrever documentações
