Plataforma de Dados : expected_cashflow_entry

Created by Alessandro Oliveira, last modified by Gabrielly Ramos on Mar 28, 2022

Nesta tabela constam as os fluxos de caixa esperados nos contratos da a55. Quando uma operação é creditada, cancelada, alterada, etc, o status do cashflow muda, conforme https://a55tech.atlassian.net/wiki/spaces/PDD/pages/1900216832.

Coluna

Conteúdo

Comentários

id

Identificador único do cashflow esperado.

active

Indica se o cashflow ainda é esperado ou não.

Todos os valores são verdadeiros.

amount

Montante do cashflow esperado (incluindo juros)

date

Data esperada do cashflow.

note

Comentários sobe o cashflow esperado.

Dado inteiramente nulo.

operation_id

Operação associada ao cashflow esperado. Referência à tabela operation.

status_id

Identificador do status do cashflow esperado. Referência à tabela https://a55tech.atlassian.net/wiki/spaces/PDD/pages/1900216832.

type_id

Identificador do tipo de operação que corresponde o cashflow. Referência à tabela https://a55tech.atlassian.net/wiki/spaces/PDD/pages/1900216843.

vehicle_id

Identificador de qual veículo está hospedado o crédito. Referência à tabela vehicle.

instrument_id

Identificador de qual contrato o cashflow esperado pertence. Referência à tabela instrument.

principal

Valor inicial do cashflow esperado (sem contar juros e taxas).
