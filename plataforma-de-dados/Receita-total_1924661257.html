Plataforma de Dados : Receita total

Created by Alessandro Oliveira (Unlicensed), last modified on Apr 13, 2022

Owner: Daniella Souza

Por receita total, entende-se tudo o que a a55 recebe de dinheiro. O cálculo base é feito considerando o valor da TAC a55 (checar TAC A55), somado ao valor dos juros referentes as parcelas pagas.

Para o cômputo do juros (JS), é feito a subtração do valor esperado a se receber, disposto na tabela https://a55tech.atlassian.net/wiki/spaces/PDD/pages/1903001635 (ece), com o valor efetivamente pago, disposto na tabela realized_cashflow_entry * (rce), de acordo com o seguinte fluxo:

  • Se o valor pago (rce.amount) for maior do que o valor esperado (ece.amount), então:

    • JS = ece.amount - ece.principal. (Por principal, entende-se o valor original da parcela, sem juros).

  • Se o valor pago (rce.amount) for menor do que o valor esperado (ece.amount), então:

    • Se o valor pago (rce.amount) for menor do que o juros esperado (ece.amount - ece.principal), então:

      • JS = rce.amount

    • Se o valor pago (rce.amount) for maior do que o juros esperado (ece.amount - ece.principal), então:

      • JS = ece.amount - ece.principal
