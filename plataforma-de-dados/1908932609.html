Propósito: Este relatório visa dar informações bancárias sobre os contratos da a55.

Entre as informações presentes, tem-se:

  • Para cada contrato, identificado pelo número do contrato, é mostrado dados do cliente, tais como nome, CNPJ e a data do desembolso.

  • Caso a amortização seja feita em parcelas, é mostrado a quantidade faltante de parcelas a serem pagas.
