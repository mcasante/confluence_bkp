Plataforma de Dados : Conversão de novos cadastros

Created by Alessandro Oliveira (Unlicensed) on Mar 02, 2022

Responsável: Karen Moraes (Unlicensed)

URL: https://metabase.a55.tech/question/2115-conversao-de-novos-cadastros

Propósito: Este relatório visa dar informações gerais sobre os clientes cadastrados na plataforma da a55.

Entre as informações disponíveis, tem-se:

  • Informações de registro do cliente, tais como CNPJ, nome, endereços, área de atuação, etc..

  • Tempo de funcionamento (fundação) do cliente.

  • Indicador se é MEI ou não.

  • Empresa parceira de originação caso haja, assim como flags de origem para os que vieram via motor de busca.

  • Receita média dos últimos três meses.

  • Desembolso requisitado.

  • Data e valor do desembolso realizado pela a55.
